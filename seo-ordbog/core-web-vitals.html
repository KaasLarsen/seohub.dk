Hvad er Core Web Vitals?

De vigtigste UX-målinger Google bruger som ranking signal.

Definition

Core Web Vitals er tre tekniske målinger for brugeroplevelse: LCP (Largest Contentful Paint), FID (First Input Delay) og CLS (Cumulative Layout Shift).

Hvorfor vigtigt?

  • Indgår i Googles Page Experience ranking signal.
  • Direkte relateret til hastighed og brugertilfredshed.
  • Bedre Vitals → bedre konvertering og lavere bounce rate.

Måling

Kan måles i PageSpeed Insights og Google Search Console under "Core Web Vitals".
