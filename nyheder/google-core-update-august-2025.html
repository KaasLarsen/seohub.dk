Google Core Update (august 2025) – de første observationer

26. aug 2025 • Google Core Update
← Tilbage til nyheder

Kort fortalt (TL;DR)

  • Størst udsving ses på YMYL og affiliate-sider.
  • Thin/duplikeret indhold ryger oftere ned – styrk kategorier og intent-match.
  • Hold øje med Core Web Vitals på mobil (INP).

Google ruller en ny Core Update ud i slutningen af august. De første data peger på, at sider med uklart intent og tynde kategorier påvirkes negativt, mens stærke topic clusters klarer sig bedre.

Fokusér på at samle overlappende indhold, opdatere ældre artikler og sikre indre linking mod dine vigtigste landingssider. Tjek også performance på mobil – især INP.

Hvad skal du gøre nu?
  • Prioritér 10–20 sider med høj business-værdi: opdater & forbedr.
  • Saml tynde/overlappende artikler → én stærkere side (301 hvis relevant).
  • Forbedr interne links (fra ordbog/blog til dine pillars).
Kilder:
  • Egnet data fra Search Console og rank-trackere (de seneste 48–72 timer).
