SERP Preview & Meta-tjek

Forhåndsvis din titel og meta-beskrivelse – og få hurtige længdeindikatorer for bedre CTR.

Hvorfor tjekke titel og meta?

Din title og meta description er det, brugeren møder i Googles søgeresultater. En skarp vinkel og korrekt længde hjælper din CTR (click-through rate). Brug vores preview til at se en Google-lignende visning og få hurtige længdeindikatorer:

  • Title: sigt efter 50–60 tegn (ikke for kort, ikke for langt)
  • Meta: sigt efter 140–160 tegn med klar værdi + CTA
  • Match søgeintention og brug tydelige H1/H2-strukturer på siden

Sådan bruger du værktøjet

  1. Skriv din titel, URL og meta-beskrivelse i felterne
  2. Se længdetællere opdatere live (grøn = godt)
  3. Justér, til det ser skarpt ud i forhåndsvisningen

Ofte stillede spørgsmål

Skærer Google altid ved samme antal tegn?
Nej. Google bruger i praksis pixelbredde, og derfor kan to titler med samme tegnantal opføre sig forskelligt. Brug vores indikatorer som guideline – og test i praksis.
Kan jeg bruge emojis i title?
Nogle emojis kan vises, men påvirker længden og kan blive filtreret. Brug dem med omtanke.
