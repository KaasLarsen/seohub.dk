On-page SEO tjekliste (2025)

En praktisk, kort tjekliste du kan bruge side for side – fra titler og meta til interne links og Core Web Vitals.

TL;DR

  • Title 50–60 tegn, stærk vinkel. Meta 140–160 tegn med USP + CTA.
  • Én H1; logisk H2/H3-struktur; svar på søgeintentionen.
  • Interne links til målsider, beskrivende ankertekster.
  • Schema (FAQ/HowTo/Article) hvor relevant.
  • Core Web Vitals: hurtig LCP, lav INP, stabil CLS.

1) Titler & meta descriptions

  • Title: 50–60 tegn, primært søgeord tidligt, unik vinkel.
  • Meta: 140–160 tegn, præcis værdi + CTA (øger CTR).
  • Undgå duplikater; hold brand i halen, hvis plads.

Tjek længder og visning i vores SERP & Meta-værktøj.

2) Headings & indholdsstruktur

  • Én H1; brug H2/H3 til at svare på under-spørgsmål.
  • Skriv skimme-venligt: korte afsnit, bullets, tabeller.
  • Brug relevante termer/synonymer naturligt.

3) Interne links

  • Link fra relaterede sider til dine vigtigste landingssider.
  • Brug beskrivende ankertekster (ikke “læs mere”).
  • Opdater 3–5 ældre sider, så de linker til nye artikler.

Få input i intern linkbuilder.

4) Struktureret data (schema)

  • FAQPage, HowTo eller Article hvor relevant.
  • Hold JSON-LD korrekt formateret og uden fejl.

5) Core Web Vitals

  • LCP: optimer hero-billeder, server-side cache/CDN.
  • INP: mindre JS, udsæt ikke-kritiske scripts.
  • CLS: reserver plads til billeder/iframes/fonts.

6) Billeder & URL’er

  • Komprimer billeder; brug korrekte dimensioner og alt-tekster.
  • Enkle, læsbare URL’er; undgå store bogstaver/space.
